Python 程式語言與其他以往的程式語言的邏輯上接近，但是程式的撰寫方式注重於排版。其編譯方式由排版的縮排架構進行解讀。

其中，包含幾個特點包含list架構，資料處理的便利性，與多元又開放的函式庫等等。

以下簡單列舉Python程式語言的大致樣貌。

```
import os

if __name__ == "__main__":
	for i in range(10):
		print(i)

```
