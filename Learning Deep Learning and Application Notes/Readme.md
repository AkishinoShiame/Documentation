### Abstract
## 概述

本文件為本人於2015年至2017年於研究所學習機器學習領域中的深度學習等相關知識領域的整理。其中盡可能的列舉出所學過的內容、包含理論的學習與理解等等，也同時將會把實作的相關成果做整理預紀錄。其中將也囊括了GPU運算之相關環境建置等等。並因應所需整理相關網站連結。
